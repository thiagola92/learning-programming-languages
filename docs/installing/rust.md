# Rust

## Linux

=== "Oficial"
    
    A linha seguinte baixa o script de instalação e o executa:  
    
    ```
    curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
    ```

=== "Ubuntu"
    
    Mantido pelos desenvolvedores do Ubuntu:  
    
    ```
    sudo apt install rust-all
    ```
