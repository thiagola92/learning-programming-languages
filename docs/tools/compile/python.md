# Python

??? info "Interpreted Language"
    
    Uma linguagem interpretada é responsável por ler e executar o código ao mesmo tempo.  

=== "Execute"
    
    Executa o código:  
    
    ```
    python main.py
    ```

    Dependendo do sistema operacional, talvez precise usar `python3`:  
    
    ```
    python3 main.py
    ```
