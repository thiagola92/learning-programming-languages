# Go

## Print

=== "Formated"

    ```go
    package main

    import "fmt"

    func main() {
        fmt.Printf("%s\n", "example")
    }
    ```

## Scan

=== "Formated"

    ```go
    package main

    import "fmt"

    func main() {
        var text string
        fmt.Scanf("%s", &text)
    }
    ```
