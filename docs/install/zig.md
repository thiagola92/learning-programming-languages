# Zig

=== "Oficial"
    
    Oficialmente não existe um instalador, você selecionaria o binário correto para o seu OS e seguiria as instruções:  
    
    [https://ziglang.org/download/](https://ziglang.org/download/)  

=== "Fedora"
    
    Mantido pelos desenvolvedores do Fedora:  

    ```
    sudo dnf install zig;
    ```
    
    !!! note
        
        Fedora armazena o binário em `/usr/bin` então não precisa adicionar no path.
