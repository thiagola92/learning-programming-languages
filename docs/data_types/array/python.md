## List

List é uma alternativa para arrays em algumas linguagens.  

```python
from array import array

a = ['i', [1, 10, 100]]

print(f"a = {a}")
```

```
a = ['i', [1, 10, 100]]
```

## Array

A linguagem utiliza a classe (`array`) para simular alguns comportamentos de um array.  

```python
from array import array

a = array('i', [1, 10, 100])

print(f"a = {a}")
```

```
a = array('i', [1, 10, 100])
```
