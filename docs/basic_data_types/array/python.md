    Esta linguagem possui uma adaptação do conceito.

## List

É uma alternativa para arrays.  

```python
from array import array

a = ['i', [1, 10, 100]]

print(f"a = {a}")
```

```
a = ['i', [1, 10, 100]]
```

## Array

A linguagem utiliza a classe (`array`) para simular alguns comportamentos de um array.  

```python
from array import array

a = array('i', [1, 10, 100])

print(f"a = {a}")
```

```
a = array('i', [1, 10, 100])
```
